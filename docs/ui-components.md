# Zangi Website – UI Components

Define the reusable UI components.

## Components Needed

- navigation bar
- hero buttons
- book cards
- section headers
- feature blocks
- testimonials
- CTA banners
- footer
- product badges
- age tags

## For Each Component Define

- purpose
- visual style
- typography
- spacing
- interaction behavior
- color usage

The component system should ensure **consistency across the website**.
