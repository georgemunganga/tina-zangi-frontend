# Zangi Website – Strategy

Define the strategic purpose of the Zangi website.

## Current Focus

Version 1 of the website should focus on:

1. Introducing the Zangi story world
2. Presenting the Zangi character
3. Selling the Zangi story book
4. Selling Zangi activity books
5. Building trust with parents

Do NOT add complex features yet such as:

- games
- ticketing systems
- learning platforms
- community systems

## Define

Please explain:

- website purpose
- primary audience
- secondary audience
- user goals
- conversion goals
- emotional experience visitors should feel

## Desired Emotional Tone

Visitors should feel:

- curiosity
- adventure
- trust
- inspiration
- cultural richness
