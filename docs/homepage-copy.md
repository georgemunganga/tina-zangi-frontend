# Zangi Website – Homepage Copy

Write homepage copy for the Zangi website.

## Audience

Primary:
- kids aged 5–17

Secondary:
- parents

## Products

- Zangi story book
- Zangi activity books

## Write Copy For

- hero section
- Zangi introduction
- featured books section
- activities section
- parent trust section
- final call-to-action section

## Tone

The writing should feel:

- adventurous
- inspiring
- warm
- culturally rich
- premium

Avoid overly childish language.
